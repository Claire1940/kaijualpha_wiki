<Section icon={Shield} title="Custos e Métodos de Desbloqueio">

Desbloquear o **kaijualpha limit break** é um marco importante que exige recursos significativos. Os desenvolvedores disponibilizaram dois caminhos principais para acessar este sistema: o caminho F2P (Gratuito para Jogar) usando Células e o caminho Premium usando Robux.

| Método | Tipo de Recurso | Custo Estimado | Ideal Para |
| :--- | :--- | :--- | :--- |
| **Grind** | Células | 50.000 - 100.000 | Jogadores ativos diários |
| **Desbloqueio Imediato** | Robux | 500 - 800 | Jogadores com tempo limitado |
| **Recompensa de Evento** | Tokens Especiais | Sazonal | Colecionadores / Completionists |

As Células são obtidas ao derrotar outros Kaijus, completar missões diárias e participar de eventos mundiais. Embora o custo de Células seja alto, ele é perfeitamente alcançável através de uma gameplay consistente. A opção de Robux é voltada para jogadores que desejam pular a fase de coleta de recursos e iniciar imediatamente sua jornada rumo ao nível 200.

<Callout type="info" title="Gerenciamento de Recursos">
Se você estiver escolhendo o caminho das Células, recomenda-se começar a economizar sua moeda a partir do nível 80. Quando atingir o nível 100, você já terá uma parte substancial das Células necessárias prontas para o upgrade.
</Callout>

O desbloqueio é permanente para aquele Kaiju específico. Se você mudar para um Kaiju diferente, precisará desbloquear o limit break para o novo personagem separadamente. Isso incentiva os jogadores a dominarem um único Kaiju "Principal" antes de partirem para outros.

</Section>
