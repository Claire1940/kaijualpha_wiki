### Heisei Godzilla
O Heisei Godzilla é amplamente considerado por jogadores experientes como a escolha ideal para o farm de cells. A principal vantagem deste personagem reside em seu kit de feixe duplo. Ao se posicionar em um terreno elevado, como a encosta de uma montanha ou uma estrutura alta, você pode disparar ambos os feixes para baixo em zonas de combate lotadas. Isso permite que você marque dezenas de jogadores ativos e inimigos de IA sem se colocar em perigo direto.

### Shin Godzilla
O Shin Godzilla é uma potência adequada para escaramuças de curto a médio alcance. Embora careça do alcance extremo do Heisei, ele compensa com ataques de área de efeito devastadores. Seu feixe traseiro característico é perfeito para situações em que você está cercado por outros jogadores. Para utilizar o Shin Godzilla de forma eficaz, procure áreas lotadas onde os jogadores estão se agrupando, mova-se para o centro e ative seu feixe traseiro para atingir todos simultaneamente.

### Base Godzilla
O Base Godzilla é o ponto de entrada padrão para novos jogadores. Embora não iguale as velocidades de farm especializadas do Heisei ou Shin, ele apresenta o Ataque Atômico Aéreo, que fornece uma área de efeito respeitável. Usar o Base Godzilla é necessário para cumprir os requisitos de nível necessários para comprar e desbloquear as variantes mais poderosas.

| Tipo de Habilidade | Heisei Godzilla | Shin Godzilla | Base Godzilla |
| :--- | :--- | :--- | :--- |
| Feixe Primário | Longo alcance, alta atividade | Transição de chama para feixe | Sopro atômico padrão |
| Ataque Secundário | Feixe secundário | Feixe traseiro / feixe de cauda | Pisoteio / golpe de cauda |
| Melhor Ambiente | Picos de montanha, longo alcance | Vales lotados, combate próximo | Campos abertos em geral |
