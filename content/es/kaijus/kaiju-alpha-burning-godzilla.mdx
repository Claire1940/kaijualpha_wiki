<Section icon={Zap} title="Análisis de Habilidades y Movimientos">

El conjunto de herramientas de **Kaiju Alpha Burning Godzilla** se basa en el "Rayo de Calor Infinito" y el "Pulso Nuclear". Dominar la rotación entre estos dos es la diferencia entre un jugador novato y uno veterano. En 2026, los desarrolladores ajustaron los tiempos de reutilización para favorecer a los jugadores que pueden alternar sus habilidades de utilidad de manera efectiva.

| Nombre de Habilidad | Cooldown | Tipo de Daño | Efecto |
| | | | |
| Garra Carmesí | 2.0s | Físico | Aplica Daño por Quemadura durante 5 segundos |
| Rayo de Calor Espiral | 8.0s | Energía | Rayo de alta penetración con retroceso |
| Pulso Nuclear | 12.0s | AoE | Limpia proyectiles cercanos y aturde |
| Rayo de Calor Infinito | 15.0s | Ultimate | Rayo masivo que se ensancha con el tiempo |
| Utilidad Meltdown | 1.0s | Buff/Debuff | Alterna el aumento de daño y el drenaje de PS |

El **Pulso Nuclear** es tu principal herramienta defensiva. Aunque Burning Godzilla carece de un escudo tradicional, el pulso puede anular ráfagas de misiles entrantes y empujar a voladores agresivos de corto alcance como Rodan o Mothra.

<Callout type="warning" title="Advertencia Crítica">
Activar la utilidad Meltdown cuando tienes menos del 15% de PS es casi siempre fatal. Usa la build de "Regeneración Sucesiva" si planeas permanecer en estado Meltdown por períodos prolongados.
</Callout>

**Estrategia de Rotación de Habilidades:**
1. Inicia con el **Rayo de Calor Espiral** para debilitar al objetivo.
2. Activa **Meltdown** una vez que se haya cerrado la distancia.
3. Usa **Garra Carmesí** dos veces para acumular el debuff de Quemadura.
4. Ejecuta el **Pulso Nuclear** si el enemigo intenta contraatacar.
5. Termina con el **Rayo de Calor Infinito** antes de desactivar Meltdown para recuperarte.

</Section>

<Section icon={Target} title="Cómo Desbloquear a Burning Godzilla">

Desbloquear al **Kaiju Alpha Burning Godzilla** requiere una serie específica de pasos de progresión. No es una unidad inicial y requiere una inversión significativa en el árbol base de Godzilla.

<Steps>
  <Step num="1" title="Alcanza el Nivel 50 con Godzilla">
    Primero debes dominar al Godzilla Heisei estándar. Esto asegura que comprendas las mecánicas básicas de movimiento y rayos antes de añadir la capa de Meltdown.
  </Step>
  <Step num="2" title="Colecciona 5,000 Células-G">
    Las Células-G se ganan a través del combate PvP y destruyendo estructuras de la ciudad. La variante Burning es una evolución premium que requiere una inversión importante de moneda.
  </Step>
  <Step num="3" title="Completa 'Despertar Volcánico'">
    Esta es una misión de desafío en solitario ubicada en el mapa de Costa Sierpe. Debes sobrevivir a un ataque de 5 minutos mientras tus PS se drenan constantemente.
  </Step>
  <Step num="4" title="Ascensión Final">
    Una vez completado el desafío, navega al Menú de Evolución y gasta las Células-G para desbloquear permanentemente la forma Burning.
  </Step>
</Steps>

<Callout type="success" title="Consejo de Desbloqueo">
Equipa módulos de "Aumento de Salud" durante el desafío Despertar Volcánico para que el temporizador de supervivencia sea mucho más fácil de gestionar.
</Callout>

</Section>
